git clone https://github.com/mdecourse/scarysim.git

修改 scarysim.pro, 蓋掉第 11 行, 成為 # LIBS     += -lglut -lGLU

cd scarysim.pro 進入後打指令 qmake -o Makefile scarysim.pro

接下來mingw32-make -f Makefile.Release

mingw32-make -f Makefile.Release 可以在 release 目錄取得 scarysim.exe

將 scarysim.exe 移至倉儲根目錄, 並且需要 msys64_20210419\mingw64\bin 路徑中的 dll 動態連結程式庫配合執行.新創一個 scarysim.bat
